Сервер нужно запускать от администратора.
Либо запускать Visual Studio от администратора и уже там запускать сервер.
Клиенты надо запускать в количестве нескольких штук из .exe файла,

Для того чтобы пользоваться чатом, нужно пройти регистрацию, нажав на кнопку «Регистрация».
Необходимо ввести свободный логин и 2 раза ввести пароль. Если всё сделано верно, регистрация пройдёт и чат будет доступен.
Если пользователь уже зарегистрирован, нужно нажать на кнопку «Уже есть аккаунт» и ввести логин и пароль.

Если закрыть форму регистрации — закроется вся программа.

Логины и пароли сохраняются в Мои Документы/GrechkaChat/UserData.json.

Когда чат станет активным, нужно нажать на кнопку «Подключиться». После этого в текстовое поле можно писать. Сообщения отправляются нажатием на кнопку «Отправить сообщение».
Также сообщения отправляются, если нажать Enter. Если зажать Shift + Enter, то текстовое поле перейдёт на новую строку.
Пустое сообщение отправить нельзя.
В правой части чата отображается количество пользователей онлайн.
Если нажать на кнопку под названием «Список», можно узнать ники пользователей онлайн.
Для обновления списка есть кнопка «Обновить». Также обновление происходит, если заново открыть окно списка.
